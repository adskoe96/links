Fluffy Commands

Voice Channel Commands

>Play [URL]

This command can play you music from youtube or a file in the voice channel, make sure you are in the voice channel.

>TTS [Ð¡ountry code] (String)

This command allows the bot to say your words in the voice channel you should be in.

>Stop

This command allows you to turn off the music to turn on another, make sure it is on.

>Pause

This command allows you to pause the music to play it later, make sure the music was playing before you paused.

>Resume

This command can continue your music, if you have paused it before.

>Retry

This command allows you to play the previous music again, if you played it before of course.

>Join

This command forces the bot to connect to the voice channel you should be in.

>Leave

This command allows the bot to exit the voice channel in which it was.



Text Channel Commands

>Start (String)

This command changes the bot's activity.

>WS [URL]

This command allows you to download the structure of the site in html format.

>QR (String)

With this command you can encrypt your words in QR code.

>Clear (Int)

This command will help you delete as many messages as you need.



Additionally

String - words.

Int - number.

URL - website address.

Country code - ru, en, etc.
